the chat

the chat room lets visitors leave messages in real-time. it uses firebase, which is like a shared notebook in the cloud that everyone can write to and read from instantly. when you send a message, it goes to firebase, and everyone's browser gets notified to show the new message.

the applications

flow garden, sticky notes, and pomodoro are mini-apps that live inside the site. they're self-contained â€” each one is its own little webpage that gets loaded into a "window" when you open it.

where it lives

the site is hosted on github pages, which is a free service that takes files from a folder and makes them available on the internet. when i make changes, i "push" the new files to github, and within a minute or two, the live site updates.

no build process

unlike most modern websites, this one has no complicated build step. there's no webpack, no react, no npm install. just plain html, css, and javascript files that your browser reads directly. it's refreshingly simple â€” like websites used to be.

the magic

the "magic" is really just clever use of basic web technologies:

  • css makes things look like windows 98 (the gray backgrounds, beveled edges, pixel fonts)
  • javascript handles dragging windows, minimizing to the taskbar, loading content
  • firebase powers the real-time chat
  • github pages hosts everything for free

that's it. no frameworks, no servers to maintain, no monthly bills. just files on the internet, like it's 1999.
