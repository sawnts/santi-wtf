the basics

notes live as simple text files on my computer. i write them in obsidian, a note-taking app. when i'm ready to share, a simple script converts them to web pages (more on the tech at the end).

how ideas connect

linking is thinking. in my notes, i can link to other notes using double brackets.

backlinks are also crucial. when note A links to note B, note B automatically knows about it and shows "what links here" at the bottom. this linking matters because:

  • ideas don't exist in isolation â€” they gain meaning through connections
  • i can stumble upon relationships i forgot about
  • i can trace how concepts relate to each other

it's like a personal wikipedia, but messier and a bit more honest.
